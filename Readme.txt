**TO RUN AS .EXE FILE:
----------------------
Step 1: Run the requirement.bat file. 
Step 2: Run the "report.exe file". The final reports are generated and stored in 
        "reports" folder.
  
--------------------------------------------------------------------------------------------
**To run as python script:
--------------------------
Step 1: Run the requirement.bat file.
Step 2: use command "python report.py" to run the script. The final reports are generated 
        and stored in "reports" folder.
